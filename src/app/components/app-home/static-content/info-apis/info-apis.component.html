APIS

API CKAN

La web de Aragón Open Data está desarrollada sobre CKAN, que es un producto de software libre creado y evolucionado por la Open Knowledge Foundation. Este producto incorpora de manera nativa una API que permite el acceso directo a los datos. Se puede encontrar toda la información de la API siguiendo este enlace.

La URI de acceso principal a la API es:
/datos/api

De forma adicional, la API de CKAN ha sido ampliada con tres operaciones más:
Para obtener el número de conjuntos de datos y de recursos almacenados en Aragón Open Data: /datos/api/getDataCount
Para obtener los cinco conjuntos de datos más recientes almacenados en Aragón Open Data: /datos/api/mostRecentDataset
Para obtener los cinco conjuntos de datos más visitados en Aragón Open Data: /datos/api/mostDownloadedDataset

API Social data

Para acceder a los servicios ofrecidos por la API de Aragón Open SocialData es necesario acceder a una URL distinta en función del servicio solicitado. Todas las URL están compuestas por dos partes, la base y los endpoints de cada servicio.

/socialdata

Un endpoint, dentro del contexto de una API REST, es un mecanismo que permite ejecutar métodos remotos y obtener los resultados de ellos, de esta manera los usuarios de la API pueden realizar consultas sobre los datos. La API de Aragón Open SocialData provee dos endpoints, uno para recoger información relacionada con las tendencias del momento en Aragón y otra para acceder al contenido sin agregar.

API AragoPedia

La web de AragoPedia está desarrollada sobre MediaWiki, que es un producto de software libre desarrollado por y para su uso por Wikimedia y sus proyectos. Está disponible libremente a otros para su uso (y modificación), y es utilizado por muchas entidades alrededor del mundo. Este producto incorpora de manera nativa una API que permite el acceso directo a los contenidos almacenados. Se puede encontrar toda la información de la API siguiendo este enlace.

API AragoDBpedia 1.0
Necesidad de credenciales

Para obtener resultados de esta API, es necesario proporcionar a las llamadas un parámetro con una credencial (api_key). Esta api_key se puede obtener de forma gratuita mediante el registro como usuario de la página 3scale.net

Por el momento, se ofrece la posibilidad de utilizar una api_key genérica y compartida, si bien en el futuro podría ser necesario el registro individualizado de todos los usuarios.

En el caso de que seas un desarrollador que vaya a hacer un uso intensivo de este servicio, rogamos contactes con nosotros a través de opendata@aragon.es para proporcionarte una api_key individualizada para tus desarrollos.

GOBIERNO ARAGÓN OPEN DATA CORE (GA_OD_Core)

GA_OD_Core se trata de un API creada con la finalidad de poder obtener datos de determinadas Vistas usadas y servidas actualmente a través del portal OpendataAragon.

¿Para que sirve?

Con esta API pretendemos dar la oportunidad a desarrolladores de consultar estos datos de forma que estén siempre actualizados y preparados para ser usados en otras aplicaciones o servicios.

¿Cómo se usa?

La URL base para acceder a los servicios de la API de Aragón OpenData GA_OD_Core es la siguiente./GA_OD_Core.

Con la tecnología del framework swagger, se muestra la documentación de manera descriptiva y estructurada. /GA_OD_Core/ui

Desde ahí podremos consultar todas las funcionalidades que posee, las cuales describimos a continuación.
